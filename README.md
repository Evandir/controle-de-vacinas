# controle-de-vacinas

Este projeto tem como objetivo implementar progressivamente e de forma didática uma aplicação web de controle de vacinas em crianças.

O frontend da aplicação foi desenvolvido com Angular e o backend foi simulado pela implementação de uma API Fake, usando o JSON Server.
